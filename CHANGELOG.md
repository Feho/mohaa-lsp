# Changelog

All notable changes to the Morpheus Script extension will be documented in this file.

## [0.1.0] - 2025-02-05

### Added
- Initial release
- Language Server Protocol (LSP) support for Morpheus Script
- Syntax highlighting via TextMate grammar
- Code completion for built-in functions, events, and properties
- Hover information with function documentation
- Go to Definition for local and cross-file symbols
- Find References for symbols across the workspace
- Document symbols outline
- Workspace symbol search
- Diagnostics for common errors
- Semantic tokens highlighting
- Inlay hints for function parameters
- Call hierarchy (incoming/outgoing calls)
- Code folding for blocks and comments
- Document links for file references
- Code lens for function references
- Rename symbol support
- Signature help for function calls
- Selection range provider
- Linked editing for paired tokens

### Supported Games
- Medal of Honor: Allied Assault (AA)
- Medal of Honor: Spearhead (SH)
- Medal of Honor: Breakthrough (BT)
- Reborn
- NightFall
